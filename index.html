Web小组分享--Github Pages

 1.What are Github Pages ?

       GitHub Pages are public webpages freely hosted and easily published through our site. You can create and publish them online using the Automatic Page Generator.

      github 设计了 Pages 功能，允许用户自定义项目首页，用来替代默认的源码列表。简单的说，github Pages 可以被认为是用户编写的、托管在 github 上的静态网页。不仅可以方便的为项目建立介绍站点，也可以用来建立个人博客。

      Github Pages有以下几个优点:

  • 免费，轻量级的博客系统，没有麻烦的配置
  • 使用标记语言，比如Markdown
  • 无需自己搭建服务器
  • 享受 git 的版本管理功能，不用担心文章遗失。
  • 你只要用自己喜欢的编辑器写文章就可以了，其他事情一概不用操心，都由 github 处理。
  • 可以绑定自己的域名
